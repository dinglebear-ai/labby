# Activity Operator Feed Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement `/activity` as an operator convenience feed that reliably shows user-visible app activity across gateways, services, devices, MCP usage, chat, artifacts, marketplace/plugins, settings, and OAuth.

**Architecture:** Keep logs as the storage and SSE transport, but add a typed activity contract in `fields_json.activity`. Backend mutation paths emit activity metadata with actor/target/scope/result, and the frontend consumes the contract first with legacy log fallback only for older events. The page uses initial search plus SSE, with filters, summary cards, diagnostics, and mock-data coverage.

**Tech Stack:** Rust 2024, axum, tracing, serde, SQLite-backed log store, Next.js, React, TypeScript, existing gateway-admin log clients.

## Implementation Notes

- Treat `/activity` as an operator convenience feed, not an immutable audit trail.
- Do not add new storage migrations in the first pass. Store activity in `fields_json.activity`.
- Backend mutations own backend activity. Frontend `recordActivity()` is only for UI-local actions that do not already hit backend mutation code.
- Do not log raw params, request bodies, file contents, diffs, tokens, cookies, OAuth codes, OAuth state, auth headers, CSRF tokens, or secret env values.
- Do not emit high-volume noise such as heartbeats, list calls, hover/tab changes, or layout toggles unless explicitly promoted to meaningful settings activity.
- Existing `handle_action()` behavior must remain backward-compatible for routes not converted in this plan.
- Follow `docs/OBSERVABILITY.md` for surface/subsystem/redaction rules.
- Use @superpowers:test-driven-development for implementation and @superpowers:verification-before-completion before completion claims.
- If using browser verification, use the `browser-devtools-cli` skill or local Chrome CDP fallback as previously verified in this repo.

pub struct ActivityRecord {
    pub schema_version: u8,
    pub kind: ActivityKind,
    pub verb: ActivityVerb,
    pub title: String,
    pub summary: String,
    #[serde(skip_serializing_if = "Option::is_none")]
    pub actor: Option<ActivityActor>,
    pub target: ActivityTarget,
    pub scope: ActivityScope,
    pub result: ActivityResult,
    #[serde(default, skip_serializing_if = "Map::is_empty")]
    pub metadata: Map<String, Value>,
    #[serde(default, skip_serializing_if = "Map::is_empty")]
    pub links: Map<String, Value>,
}

impl ActivityRecord {
    pub fn to_fields_json(&self) -> Value {
        json!({ "activity": self })
    }
}

pub fn actor_from_auth_context(auth: Option<&AuthContext>) -> Option<ActivityActor> {
    auth.map(|ctx| ActivityActor {
        subject: ctx.sub.clone(),
        email: ctx.email.clone(),
        issuer: ctx.issuer.clone(),
        via_session: ctx.via_session,
    })
}

#[derive(Clone, Copy, Debug, PartialEq, Eq)]
pub struct ActivityClassification {
    pub kind: ActivityKind,
    pub verb: ActivityVerb,
    pub target_type: &'static str,
    pub title: &'static str,
}

pub fn classify_action(service: &str, action: &str) -> Option<ActivityClassification> {
    let normalized = action.strip_prefix(&format!("{service}.")).unwrap_or(action);
    match normalized {
        "gateway.add" | "add" => Some(row(ActivityKind::Gateway, ActivityVerb::Added, "gateway", "Added gateway")),
        "gateway.remove" | "remove" => Some(row(ActivityKind::Gateway, ActivityVerb::Removed, "gateway", "Removed gateway")),
        "gateway.update" | "update" => Some(row(ActivityKind::Gateway, ActivityVerb::Updated, "gateway", "Updated gateway")),
        "gateway.mcp.enable" => Some(row(ActivityKind::Gateway, ActivityVerb::Enabled, "gateway", "Enabled gateway MCP")),
        "gateway.mcp.disable" => Some(row(ActivityKind::Gateway, ActivityVerb::Disabled, "gateway", "Disabled gateway MCP")),
        "gateway.virtual_server.enable" => Some(row(ActivityKind::Service, ActivityVerb::Enabled, "service", "Enabled service")),
        "gateway.virtual_server.disable" => Some(row(ActivityKind::Service, ActivityVerb::Disabled, "service", "Disabled service")),
        "gateway.service_config.set" => Some(row(ActivityKind::Settings, ActivityVerb::Changed, "service_setting", "Changed service settings")),
        "gateway.server.install" => Some(row(ActivityKind::Registry, ActivityVerb::Installed, "registry_server", "Installed registry server")),
        "sources.add" | "marketplace.sources.add" => Some(row(ActivityKind::Marketplace, ActivityVerb::Added, "marketplace", "Added marketplace")),
        "sources.remove" | "marketplace.sources.remove" => Some(row(ActivityKind::Marketplace, ActivityVerb::Removed, "marketplace", "Removed marketplace")),
        "plugin.install" | "marketplace.plugin.install" => Some(row(ActivityKind::Plugin, ActivityVerb::Installed, "plugin", "Installed plugin")),
        "plugin.uninstall" | "marketplace.plugin.uninstall" => Some(row(ActivityKind::Plugin, ActivityVerb::Removed, "plugin", "Removed plugin")),
        "plugin.save" | "marketplace.plugin.save" => Some(row(ActivityKind::Artifact, ActivityVerb::Edited, "artifact", "Edited artifact")),
        "plugin.deploy" | "marketplace.plugin.deploy" => Some(row(ActivityKind::Artifact, ActivityVerb::Deployed, "artifact", "Deployed artifact")),
        "plugin.cherry_pick" | "marketplace.plugin.cherry_pick" => Some(row(ActivityKind::Artifact, ActivityVerb::Forked, "artifact", "Forked artifact")),

        _ => None,
    }

```rust
fn matches_activity_query(event: &LogEvent, query: &LogQuery) -> bool {
    let activity = event.fields_json.get("activity");
    if query.activity_only && activity.is_none() {
        return false;
    }
    if let Some(activity) = activity {
        if !query.activity_kinds.is_empty() {
            let kind = activity.get("kind").and_then(Value::as_str);
            if !kind.is_some_and(|kind| query.activity_kinds.iter().any(|k| k == kind)) {
                return false;
            }
        }
        if let Some(expected) = &query.actor_subject {
            let subject = activity.pointer("/actor/subject").and_then(Value::as_str);
            if subject != Some(expected.as_str()) {
                return false;
            }
        }
        if let Some(expected) = &query.target_type {
            let target_type = activity.pointer("/target/type").and_then(Value::as_str);
            if target_type != Some(expected.as_str()) {
                return false;
            }
        }
        if let Some(expected) = &query.target_id {
            let target_id = activity.pointer("/target/id").and_then(Value::as_str);
            if target_id != Some(expected.as_str()) {
                return false;
            }
        }
    }
    true
}
```
